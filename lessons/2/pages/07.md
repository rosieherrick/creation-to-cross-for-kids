## God is Trinity

|  | TRINITY |  |
| :---: | :---: | :---: |
| "tri-" = three |   | "unit" = one |
| ![alt text](image-1.png) |   | ![alt text](image-2.png) |

***

We use the word “Trinity” to describe these three persons who are the one Eternal God.

The Word “Trinity” is made from “tri-”, meaning three (as in a tricycle) and “unit”, meaning one – that is, three-in-one.
